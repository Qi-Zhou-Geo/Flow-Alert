## Read Me
Welcome to **Flow-Alert**! <br>
If you're interested in leveraging **machine learning** and **seismic signals** for **channelized flow warning**, <br>
including debris flows, glacial lake outburst floods, and lahars, <br>
you've come to the right place. Please check out our repository!

### 1, Repository Structure <br>
```bash
Flow-Alert
├── config # configuration files
├── functions # all functions that need to be called
├── data # seismic features that calculated from raw data
└── trained_model # trained model
```
